# 📝 Todo-List React App

A simple and intuitive task management application built with React.js.

---

## ✨ Features

- ✅ Create a task  
- 🗑️ Delete a task  
- 🔼 Move a task up  
- 🔽 Move a task down  

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/Bristhis25/Todo-List.git
cd Todo-List
```

2. Install dependencies:
```bash
npm install
```

3. Start the application:
```bash
npm start
```
